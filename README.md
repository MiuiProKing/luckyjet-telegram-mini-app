# LuckyJet Exact PRO — Telegram Mini App

Веб-версия собрана из `LuckyJetExact-PRO-v2.0-FIXED-unsigned.ipa` и сохраняет
главную страницу, Lucky Jet, Aviator, Rocket Queen, обе PRO-страницы, изображения,
темы, локальную историю и текущие API-подключения.

## Telegram

Проект подключает официальный Telegram Web Apps SDK, разворачивает Mini App на
доступную высоту, учитывает safe-area и открывает внешние игровые ссылки через
Telegram.

После публикации GitHub Pages укажите адрес главной страницы в BotFather:

1. `/mybots` → выберите бота.
2. `Bot Settings` → `Configure Mini App` или `Menu Button`.
3. Вставьте HTTPS-адрес GitHub Pages, оканчивающийся на `/`.

## Локальный запуск

Откройте `index.html` напрямую или запустите любой статический HTTP-сервер в этой
папке.
