🤖 BETA AI 24/7 — установка

Выполни три шага один раз. После этого Supabase будет получать коэффициенты и сохранять историю каждую минуту, даже когда Telegram закрыт.

1Создай Edge Function

Открой Supabase → Edge Functions → Create function. Название функции: beta-ai-worker. Удали пример кода и вставь код ниже, затем нажми Deploy.

ОТКРЫТЬ EDGE FUNCTIONS

2Добавь секрет

В Edge Functions → Secrets создай секрет BETA_AI_CRON_SECRET. Значение придумай сам, например длинную строку из букв и цифр. Запомни её — в SQL нужно вставить точно такую же.

Не присылай секрет сюда. Пример имени: BETA_AI_CRON_SECRET. Значение: своя длинная случайная строка.
ОТКРЫТЬ SECRETS

3Запусти SQL

Скопируй SQL. Перед запуском замени текст YOUR_CRON_SECRET на секрет из второго шага. Затем вставь всё в SQL Editor и нажми Run.

ОТКРЫТЬ SQL EDITOR

✓Проверка

После запуска подожди 2–5 минут и открой в приложении: PRO → BETA AI 🤖. В истории должны появиться коэффициенты. Если функция открыта в Supabase, нажми Invoke — ответ должен содержать ok: true или DUPLICATE_ROUND.
