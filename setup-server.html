Активация сервера ключей
Одна Edge Function без GitHub-токена
1

Скопируй готовый код

Нажми кнопку ниже. Полный серверный код функции allpredictor-api скопируется в буфер.

2

Открой Edge Functions

В Supabase нажми Deploy a new function → Via Editor. Название функции укажи точно:

allpredictor-api

ОТКРЫТЬ SUPABASE
3

Вставь и опубликуй

Удали пример кода в редакторе, вставь скопированный код и нажми Deploy function. Обычно публикация занимает 10–30 секунд.

4

Отключи встроенную JWT-проверку

Открой настройки функции allpredictor-api и выключи пункт Verify JWT или Enforce JWT verification. Без этого новый ключ sb_publishable_… может получить ошибку 401. Безопасность останется: функция сама проверяет подпись Telegram и твой Telegram ID.

После установки

Вернись в мини-приложение → нажми 🔑 ADMIN. Появятся статистика, создание ключей, выбор 1–2 устройств, продление, отзыв лицензии и блокировка пользователей.

ВЕРНУТЬСЯ В ADMIN
